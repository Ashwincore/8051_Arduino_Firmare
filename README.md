# 8051_Arduino_Firmare
To dump Code into 8051


First connect the 8051 as shown in Figure(8051 Connection Diagram)
Now Connect the Following pins
8051            Arduino
==========================
Reset (Pin 9) =  Pin 10

MOSI (Pin 6) =   Pin 11

MISO (Pin 7) =   Pin 12

SCK (Pin 8) =    Pin 13


For Commands Refer to Arduino8051firmare.txt
